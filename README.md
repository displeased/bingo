# bingo

A simple Python script and Makefile to generate a bingo board.

## Description

For a given set of terms, generates a bingo board using pandoc templating and LaTeX rendering.

## Getting Started

### Dependencies

* xelatex
* python
* pandoc
* FreeSans Font

### Installing

* Install dependencies
* Place your terms into a file named `terms` in the repo folder

### Executing program

* Clone the git repository
* Change directory into the project directory
```
cd bingo/
```
* Run the `make` command
```
make
```
* `bingoboard.pdf` is your final result
